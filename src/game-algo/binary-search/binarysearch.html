Tebak Angka Rahasia

Kembali

Aturan Main

1. Tebak angka rahasia antara 1 dan 100.

2. Masukkan tebakan di kotak input, lalu klik "Tebak".

3. Petunjuk akan muncul (terlalu tinggi/rendah).

4. Kamu punya 7 kesempatan. Jika habis, game over.

5. Jika tebakan benar, kamu menang!

Panduan Main

1. Klik "Tebak" setelah memasukkan angka.

2. Perhatikan petunjuk di grid kanan.

3. Gunakan logika binary search untuk cepat menang.

4. Reset untuk memulai ulang permainan.

5. Kembali ke daftar game dengan tombol "Kembali".
