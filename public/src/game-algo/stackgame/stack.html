Level: 1

Aturan Main

1. Ascak dengan ukuran acak muncul di grid kiri.

2. Tekan "Push" untuk menumpuk kotak jika ukurannya lebih kecil dari kotak teratas.

3. Tekan "Pop" untuk menghapus kotak teratas jika diperlukan.

4. Skor bertambah (+10) untuk setiap push yang valid.

5. Capai skor 50 sebelum waktu habis (60 detik) untuk menang!

6. Game over jika push tidak valid, tumpukan penuh, atau waktu habis.

Panduan Main

1. Perhatikan ukuran kotak di grid kiri.

2. Gunakan "Push" untuk menumpuk kotak yang lebih kecil.

3. Gunakan "Pop" untuk menghapus kotak jika tidak sesuai.

4. Level meningkat setiap 20 poin, membuat ukuran kotak lebih ketat.

5. Reset untuk memulai ulang permainan.
