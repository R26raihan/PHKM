Aturan Main

1. Mulai dari kotak hijau dan capai kotak merah.

2. Klik kotak di sekitar untuk bergerak (atas, bawah, kiri, kanan).

3. Hindari kotak hitam (rintangan) dan pilih jalur yang benar.

4. Jika salah jalur atau menabrak rintangan, game over.

5. Saat game over, jalur terpendek akan ditampilkan.

Panduan Main

1. Klik "Mulai Main" untuk memulai permainan.

2. Klik kotak di grid kanan untuk membentuk jalur Anda.

3. Kotak hijau adalah start, kotak merah adalah tujuan.

4. Jika menang, Anda akan mencapai harta karun!

5. Gunakan "Reset" untuk memulai ulang permainan.
